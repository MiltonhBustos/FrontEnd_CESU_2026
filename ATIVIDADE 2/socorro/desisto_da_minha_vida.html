Conteúdos

  • O que é Lógica de proramação?

      Lógica de programação é a técnica de encadear pensamentos e estruturar instruções algoritmos de forma ordenada e clara para resolver um problema específico via computador

  • O que é um algoritmo?

      É uma sequência finita e ordenada de passos lógicos e não ambíguos

  • O que é um programa?

      É a codificação de um algoritmo em diferentes linguagens

Operadores

Operadores em C são símbolos especiais que instruem o compilador a realizar manipulações matemáticas, lógicas ou relacionais específicas em dados (variáveis ou constantes)

Operadores Relacionais

  • Nome Operador
    Menor que x < y
    Maior que x > y
    Menor ou igual x <= y
    Maior ou igual x >= y
    Igual x == y
    Diferente x != y

Operadores de Aritmética

  • Nome Símbolo

    Subtração -
    Multiplicação *
    Divisão /

Operadores Lógicos

  • Nome Símbolo
    AND/E &&
    OR/OU ||
    NOT/NÃO !

Laços de Repetição

Laços de repetição ou loops são estruturas de controle na programação que permitem executar um bloco de código várias vezes consecutivas, sem precisar reescrevê-lo

Laço FOR

O laço for em C é uma estrutura de repetição usada quando você sabe quantas vezes quer repetir um bloco de código

Exemplo de for

Laço WHILE

O laço while usado quando você não sabe exatamente quantas vezes o código vai repetir, mas sabe qual condição precisa ser verdadeira

Exemplo de while

Laço DO WHILE

O do while é parecido com o while, mas tem uma diferença importante, ele executa o bloco pelo menos uma vez, mesmo que a condição seja falsa

Exemplo de do while
